# 🍷Simple Gala Night Lucky Draw

A stylish browser-based lucky draw system designed for events, company parties, and gala dinners.

Live Demo  
 https://AFYQ99.github.io/Simple-Gala-Lucky-Draw/

---

## ✨ Features

- 🎰 Rolling lucky draw animation
- 🎉 Confetti celebration
- 🌌 Floating particle background
- 🎯 Optional fixed winner mode
- 👀 Hidden admin panel
- 📺 Big screen projection mode
- 🏆 Multiple prize categories
- 📜 Final winner summary page

## 🚀 Deployment

You can deploy instantly using **GitHub Pages**.

Steps:

1. Upload project
2. Go to **Settings → Pages**
3. Select branch `main`
4. Save

https://USERNAME.github.io/gala-lucky-draw/
